Script Editor

Turn a route drawn on the map into a walk script, or paste a script to draw it back. Draw a line with the toolbar (top-right of the map), pick your platform, then Export to generate the script — or paste one below and hit Import to see its path on the map.

Changelog
Recent changes
  • Rebranded to SROCodexMap with the SROCodex desert theme.
  • Removed the third-party CDN scripts and analytics tag; libraries are now served from this site.
  • Deep links (?x&y or ?x&y&z&region) drop a pin on the linked spot.
  • Added a live cursor coordinate readout (bottom-right).
  • Double-clicking a spot drops a marker there with its coordinates, a copy link, and a remove option.
  • The bottom-right label now shows the current map layer instead of an external link.
  • Added a distance measurement tool (ruler button): click points to measure a path, Esc to finish.
  • The drawing tools (top-right) now have tooltips explaining what each one does.
  • Added Layers toggles in the sidebar to show or hide NPCs and teleporters.
  • Added Monster Spawns: pick a monster to see the area where it spawns, mapped from its in-game spawn points. Town areas are cut out of the zone.
  • Monster Spawns: browse Uniques or a chosen zone — it zooms in and lists the mobs in a colour legend; click one to see where it spawns.
Planned
  • Monster spawns for dungeon monsters (currently world only).
  • Text search for NPCs and areas.
